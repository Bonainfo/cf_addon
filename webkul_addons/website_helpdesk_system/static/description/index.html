Website Helpdesk & Support System

Website Helpdesk & Support System is used to provide information and support to the end users/customers regarding the products and services of the organization.

Basic Features
  • Using this app admin can easily manage their Helpdesk and Support System.
  • Only register customers can raise tickets.
  • Helpdesk manager has to create teams and manage tickets based on team.
  • Ticket can be assigned to only to the team members.
  • Manage category on tickets.
  • Manage Topics on ticket based on category.
  • Manage ticket status.
  • Ticket creation mail notification to the customers.
  • Customer can post comment or query in the ticket.
  • Helpdesk user can post the internal note in the ticket.
  • Automatic review mail sent to the customer when ticket gets resolved.
  • Enable/disable to add reCAPTCHA in form.
  • Supports both Community and Enterprise.

  • Backend View
