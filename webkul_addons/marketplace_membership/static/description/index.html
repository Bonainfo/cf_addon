Odoo Marketplace Membership
Manage your marketplace seller with marketplace membership plan features

Note: This module is an addon for Odoo Multi Vendor Marketplace module. The price reflected here includes all dependent module's cost as well.

Features:

  • Admin can create membership plans from backend
  • Admin can manage visibility on website by publishing/un-publishing membership plan
  • Admin can set price for membership plans
  • Membership plan restrict the seller for creating/adding products
  • Seller will buy the plan as per the requirement and will pay for it
  • Seller can upgrade the plans whenever it is required
  • Admin can cancel the membership plan of any seller at any time
  • Separate page for displaying marketplace membership plan on website

  • Website View

  • Configuration
